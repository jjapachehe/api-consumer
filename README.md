
# API Consumer

This program test the APIs given as parameter and return the response code.


## Run Locally

Clone the project

```bash
  git clone git@git.rosetta.ericssondevops.com:jhon.apache/api-consumer.git
```

Go to the project directory

```bash
  cd api-consumer
```

Execution

```bash
  main.py -f apis.lst
```
